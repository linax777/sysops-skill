# 維運知識庫 (SysOps Knowledge Base)

本知識庫遵循「漸進式揭露 (Progressive Disclosure)」與「物件與任務導向」設計。
請從以下主要大類開始查詢：

- 目錄 `01_triage_rules/` : 存放分流規則、告警 ID 映射檔 (.base) 與初步判斷邏輯。
- 目錄 `02_procedures/` : 存放標準作業流程 (SOP)、故障排除與系統變更手冊。
- 目錄 `03_system_context/` : 存放深度的系統細節、資源清單、設定檔參數與架構圖。
- 目錄 `04_atomic_assets/` : 存放單獨的腳本、Prompts 與可重複使用的組件。
